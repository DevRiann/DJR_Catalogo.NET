# 📺 Gerenciador de Séries e Animes

Um sistema web desenvolvido em **.NET 10** para ajudar usuários a organizarem suas maratonas, marcando exatamente o último episódio assistido de cada série ou anime para nunca mais se perderem.

---

## 🚀 Funcionalidades Principal (CRUD Completo)

O sistema oferece um gerenciamento total das produções com as seguintes operações:
*   ➕ **Adicionar (Create):** Cadastrar novas séries ou animes na lista de acompanhamento.
*   🔍 **Visualizar (Read):** Listar todas as produções salvas e acompanhar o progresso atual.
*   ✏️ **Atualizar (Update):** Alterar o nome da produção ou atualizar o número do último episódio assistido.
*   🗑️ **Excluir (Delete):** Remover títulos da lista quando desejar.
*   📧 **Envio de Formulários:** Integração de segurança que envia as atualizações de formulários direto para o Gmail configurado via API.

---

## 🛠️ Tecnologias Utilizadas

*   **Ambiente de Desenvolvimento:** .NET 10 (ASP.NET Core Web App) 💻
*   **Banco de Dados:** SQLite (Armazenamento local rápido e leve) 🗄️
*   **Mapeamento de Dados:** Entity Framework Core (EF Core) como motor de persistência ⚙️
*   **Comunicação/E-mail:** Google API Integration & MailKit ✉️
*   **Interface Visual:** Bootstrap (Design responsivo integrado) 🎨
*   **Interatividade:** JavaScript (Destaque dinâmico de navegação no Menu) 🌐
